# Sistema de Gestión Financiera – ODA

Aplicación web desarrollada con **Django** para gestionar de forma sencilla los **ingresos y gastos de una asociación**.

Permite al tesorero registrar movimientos financieros, visualizar el balance y generar reportes exportables para el seguimiento de cuentas.

## Funcionalidades

### Gestión de movimientos
- Registro de ingresos y gastos.
- Edición y eliminación de movimientos.
- Control estructurado de transacciones.

### Balance financiero
- Cálculo automático del saldo actual.
- Visualización clara de movimientos financieros.
- Filtrado por rangos de fecha.

### Exportación de informes
- Exportación de seguimiento en **Excel (.xlsx)**.
- Generación de **informes en PDF**.

### Multiusuario
- Sistema de autenticación.
- Cada usuario visualiza únicamente sus propios datos.
